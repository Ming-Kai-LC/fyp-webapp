About This System

This COVID-19 Detection System is a Final Year Project (FYP) developed for TAR UMT's Bachelor of Data Science program. The system utilizes state-of-the-art deep learning models to analyze chest X-ray images and provide multi-model predictions with explainability features.

Key Features:

  • Spotlight Feature #1: Multi-model comparison across 6 architectures
  • Spotlight Feature #2: Explainable AI with Grad-CAM and attention visualization
  • CLAHE preprocessing for enhanced image quality
  • Role-based access control (Admin, Doctor, Patient)
  • Comprehensive admin panel for system management
  • Memory-optimized for RTX 4060 8GB VRAM
